## 最终模型

- 选定模型：随机森林（random_forest）
- 理由：在相同训练/测试集上，随机森林准确率（约87.3%）和精确率（约58.4%）最高，流失召回率（约85.8%）与另两个模型接近，综合表现最优且比单棵决策树更稳定。
